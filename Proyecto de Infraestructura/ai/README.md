# Módulo de IA / Detección con cámara (futuro)

Reservado para la fase futura del proyecto: detección automática de patentes desde una cámara
usando visión por computadora.

La idea es que este módulo capture la imagen, reconozca la patente (OCR / modelo de detección)
y llame al mismo endpoint del backend (`GET /api/patentes/:patente`), reutilizando toda la
lógica de base de datos y API externa ya existente.

**No implementado todavía.**
